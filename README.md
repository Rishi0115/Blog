# Blog App

A simple blog built with Next.js 14 and Tailwind CSS.

## Run the app

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project structure

- `app/` — pages and layouts (App Router)
- `components/` — Navbar, PostCard
- `lib/` — shared data and utilities
